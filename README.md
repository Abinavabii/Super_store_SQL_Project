# 🛒 Super Store Sales SQL Project

## 📌 Project Overview

This project demonstrates SQL-based data analysis on a Super Store Sales database. It covers database creation, table relationships, data validation, business analysis, aggregate functions, joins, subqueries, and window functions.

---

## Database Schema

The database contains four tables:

- Customers
- Products
- Orders
- Order_Items

## Business Insights

- Furniture generated the highest sales.
- Furniture category also produced an overall loss.
- High sales do not always mean high profit.
- High-profit products should receive more focus.
- Low-profit products require pricing and profitability improvements.
